# 📌 上山課程行前確認清單（GitHub 上線版）

本專案提供一份「**上山課程行前確認清單**」，方便學員於出發前自行檢查是否準備完成。
可搭配 GitHub Pages 使用，支援電腦與手機瀏覽（iPhone / Android）。

---

## 📣 重要提醒（僅提醒，不需勾選）

1. **務必於 1/5（週一）18:00 前**於 **Line 群組回報是否參加**。
   取消上山之學員，費用將轉為 **護持寶雲寺基金**，**不另行退費**。

2. **集合資訊**

   * 集合時間：**AM 05:30**
   * 集合地點：**寶雲寺對向上車**

3. **餐飲說明**

   * 本次課程提供：**早餐、午齋、藥石**
   * 請學員自備 **環保餐具（午餐用）**
   * 請自備 **空的便當盒（藥石用，需有蓋可扣緊，並貼上「組別＋姓名」）**

## 🌐 使用方式（GitHub Pages）

1. 將本專案上傳至 GitHub Repository
2. 於 **Settings → Pages** 啟用 GitHub Pages
3. 使用 `main / root` 作為來源
4. 取得公開網址，分享給學員使用
